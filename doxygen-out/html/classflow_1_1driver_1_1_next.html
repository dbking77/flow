Detailed Description

template<typename DispatchT, typename LockPolicyT = NoLock, typename ContainerT = DefaultContainer<DispatchT>>
class flow::driver::Next< DispatchT, LockPolicyT, ContainerT >

Captures the next oldest data element.

Establishes a sequencing range with range.lower_stamp == range.upper_stamp equal to the captured element stamp. Removes captured element from buffer.

Template Parameters

LockPolicyTa BasicLockable (https://en.cppreference.com/w/cpp/named_req/BasicLockable) object or NoLock or PollingLock
